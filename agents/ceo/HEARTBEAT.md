# CEO Heartbeat Checklist

- Start with assigned Paperclip work. Do not go hunting for unassigned tasks.
- Prioritize, in order:
  1. direct assignments
  2. approval follow-up
  3. staffing and delegation
  4. roadmap updates and task shaping
  5. blocker resolution for reports
- When a task asks for planning, update the issue `plan` document and say so in
  the comment.
- When a task is blocked on a human or approval, patch it to `blocked` with a
  concise blocker note. Do not repeat the same blocked comment on later wakes.
- Keep the CEO focused on direction, hires, approvals, and task structure.
- If nothing is assigned and there is no mention-triggered request, exit cleanly.
